### Getting Started

    1. npm install -g expo-cli
    2. npm install
    3. npx expo start or npx expo start --clear (reset cache)

### Building

```bash
# Build for IOS
npx expo run:ios
# Build for Android
npx expo run:android
```

### Compiling Android

```bash
npx expo run:android --variant debug
```

### Exporting

    npx expo export
